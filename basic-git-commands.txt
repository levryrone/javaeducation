git clone https://github.com/levryrone/javaeducation.git - склонувати репо локально собі

git branch                                        - подивитись список гілок і поточну

git branch -a                                     - подивитись всі гілки, в тому числі і на віддаленому репо

git checkout -b my-new-branch-name                - створити нову і перейти на неї

git add . - додати всі зміни в коміт

git commit -m "My commit message"                  - створити коміт

git push                                           - запушити

git push --set-upstream origin remote-branch-name  - запушити в конкретну гілку

Стягнутися з віддаленого репо:

git fetch                                          - витягнути посилання на віддалені гілки

git pull                                           - синхронізуватись з відаленою гілкою, стягнути її

git merge branch-to-merge-name                     - залити гілку branch-to-merge-name в мою поточну

Подивитись статус: 

git status                                         - статус змін (де я, які зміни закомічені, які ні)

git log                                            - історія комітів




git stash                                          - сховати у схованку

git stash apply                                    - дістати із схованки
